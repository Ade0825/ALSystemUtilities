ALSystemUtilities

This library provides a list of 80 methods to get every kind of system information!

View project on GitHub

Useful

The library is useful to check and get every kind of information about the device. You can embed in in every project, it is very light!

Updated

The framework is updated every time Apple presents new products, so you are always up-to-date with the informations about the hardware and battery.
